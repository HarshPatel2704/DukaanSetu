# DukaanSetu – MERN E-commerce Platform for Offline Shopkeepers

DukaanSetu is a modern **multi-role MERN stack marketplace** designed to connect **offline local shopkeepers** with **online customers**.  
The platform helps traditional businesses grow digitally by enabling product listings, online ordering, and role-based dashboards.

---

# Live Vision

> Empower local stores with digital tools and help customers shop nearby with convenience.

---

# Key Features

## Authentication & Role Management
- Secure JWT-based Login / Signup
- Password encryption using BcryptJS
- Role-based access:
  - Customer
  - Shopkeeper
  - Admin

---

## Customer Panel
- Browse all products
- Search and filter products
- Add to cart
- Place orders
- View order history
- Give ratings & reviews

---

## Shopkeeper Dashboard
- Add / Edit / Delete Products
- Manage inventory
- Track customer orders
- Mark orders as Delivered
- View sales & earnings

---

## Admin Panel
- Manage users
- Manage product categories
- Monitor platform activity
- Track popular products
- Moderate products

---

# Unique Concept

DukaanSetu solves a real-world problem:

Small offline shops often lack digital presence  
Customers prefer online convenience  

DukaanSetu bridges both worlds.
